This Adobe Illustrator template, named "separations.ai," is a standard registration and separation sheet designed for screen printing. It's structured as a single document containing five distinct artboards, each dedicated to a specific spot colour separation.

**Template Description:**

*   **Structure:** A single Adobe Illustrator (.ai) document featuring five vertically arranged artboards.
*   **Artboards:** Each artboard measures 24x36 inches (trim size) and includes a 0.125-inch bleed around all edges, resulting in an effective canvas size of 24.25x36.25 inches per artboard.
*   **Labels:** Each artboard is clearly labelled at the top with its respective spot colour: "WHITE UNDERBASE," "MARIAN BLUE," "ANTIQUE GOLD," "BRICK RED," and "INK BLACK."
*   **Marks:** All artboards feature comprehensive crop and registration marks. These include standard crosshair registration marks at all four corners of the trim box, at the midpoint of each side of the trim box, and a central registration mark, totaling nine marks per artboard.
*   **Placeholders:** Below each artboard's trim area, there is a dedicated text slot placeholder: "INK NAME + MESH COUNT," allowing for easy input of specific ink details and mesh specifications for each separation.
*   **Styling:** The template adheres to a functional, high-contrast print shop standard, utilizing clear black vector lines for all marks and text, ensuring readability and accuracy on film positives.
