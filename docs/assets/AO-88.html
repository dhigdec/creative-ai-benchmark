**Template File Description: `ridgeline_crest.ai`**

This Adobe Illustrator (.ai) document serves as the production-ready vector master for the Ridgeline Crest emblem, intended primarily for patch manufacturing. It features vector geometry hand-assembled from the traced `ridgeline_crest_master.svg` and its 4-flat-ink spot-separated counterpart, `ridgeline_crest_spot_separated.svg`.

**Structure & Content:**
The file contains multiple artboards, each precisely sized for typical 3-6 inch patch dimensions (e.g., 4x4 inches). These artboards host the complete vector paths of the emblem, presenting both the master version and its pre-separated spot color variations. No new artwork is drawn within this file; its purpose is to consolidate and arrange the traced paths for final export.

**Layers & Placeholders:**
The artwork is structured with vector paths organized for clean output and color separation. The emblem design is complete, and the file contains no empty placeholders or editable text fields for new content.

**Dimensions:**
Artboards are configured for physical output dimensions ranging from 3 to 6 inches, suitable for various patch sizes.

**Brand Styling:**
The emblem's styling is defined by a specific four-spot-color palette, critical for consistent manufacturing. These are implemented as named swatches within the document:
*   **Spot Color 1:** `#2E5A3A` (Dark Green)
*   **Spot Color 2:** `#F2E9D0` (Off-White/Cream)
*   **Spot Color 3:** `#6B7078` (Mid-Grey)
*   **Spot Color 4:** `#111111` (Black)
