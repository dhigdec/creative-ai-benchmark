**Template File:** `nimbadesk_sellsheet.indd`

**Description:**
This is an Adobe InDesign template for a one-page B2B SaaS product sell sheet, formatted for US Letter (8.5" x 11") in portrait orientation. It features a clean, modern grid layout for a professional aesthetic.

**Structure & Layout:**
The template is organized into distinct sections:
1.  **Header:** Branded section containing the company logo frame and key product identifiers.
2.  **Hero Image:** A prominent image frame (16:10 ratio) intended for a product screenshot.
3.  **Company Advantage:** A dedicated block titled 'Why teams switch to Nimbadesk', detailing company benefits.
4.  **Core Features:** A three-bullet list highlighting primary product features.
5.  **Call to Action (CTA) Band:** A prominent footer band for the main call to action.

**Layers:**
The document is typically structured with a 'Background/Graphics' layer for fixed brand elements and a 'Content' layer for editable text and image frames.

**Placeholders:**
The template contains genuine InDesign data-merge placeholder fields for dynamic content population:
*   `<<tier_name>>` (in header)
*   `<<tagline>>` (in header)
*   `<<price_line>>` (within product details)
*   `<<feature_1>>`, `<<feature_2>>`, `<<feature_3>>` (for the core features list)
*   `<<cta_line>>` (in the CTA band)
Additionally, dedicated frames are present for the company logo (header) and the hero product screenshot.

**Brand Styling:**
The design adheres to the Nimbadesk brand guidelines, utilizing a palette of #0E1F3D (navy), #3B82F6 (signal blue), and #F5F7FA (fog-grey) for a consistent and modern appearance. Typography is clean and legible, complementing the grid-based layout.
