**Template File:** `aurelia_label_template.indd` (Adobe InDesign Label Template)

**Dimensions:** 70x90mm with a 3mm bleed on all sides.
**Color Mode:** CMYK.

**Structure & Layers:**
The template is structured with distinct layers for text and graphics to facilitate editing and data-merge.
*   **Text Layer:** Contains the dynamic data-merge placeholders.
*   **Graphics Layer:** Holds the static design elements.
*   **Background Layer:** For brand color application.

**Placeholders (Data-Merge Fields):**
The template is authored with genuine InDesign data-merge fields, ready for binding with an external data source:
*   `<<product_name>>`
*   `<<key_actives>>`
*   `<<volume>>`
*   `<<batch>>`

**Typographic Hierarchy:**
A defined typographic hierarchy is applied to the merge fields:
*   `<<product_name>>`: Large, prominent serif typeface.
*   `<<key_actives>>`: Medium-sized, legible sans-serif.
*   `<<volume>>` & `<<batch>>`: Smaller, clean sans-serif for secondary information.

**Graphic Elements:**
*   An accurately positioned image frame is included on the left-hand side, designated for a "leaf side-panel texture" graphic.

**Brand Styling:**
*   **Swatches:** Dedicated brand swatches are pre-defined in the document:
    *   `Aurelia_Green` (CMYK value specified within InDesign)
    *   `Aurelia_Cream` (CMYK value specified within InDesign)
*   These swatches are applied to background elements and accent typography.

**Key Feature:** This is a natively authored InDesign `.indd` file with live, binding `<<fields>>` for data-merge, fully compliant with the `[T]` mode requirement for variable data printing.
