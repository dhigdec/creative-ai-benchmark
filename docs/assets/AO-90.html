**Template Description:**

**File Name:** `decal_sheet_template.ai`

**Purpose:** Adobe Illustrator template for a multi-decal die-cut vinyl sheet layout, meticulously authored for automated vector data merge and precise cutting. This template ensures proper binding of vector merge data to cut paths, critical for print-shop production.

**Dimensions:**
*   **Artboard:** 24" x 18" (standard sheet size for vinyl printing and cutting).
*   **Decal Cells:**
    *   **Large (1x):** Approx. 10" x 8" (e.g., window/glass decal).
    *   **Medium (2x):** Approx. 4" x 4" each (e.g., box labels).
    *   **Small (6x):** Approx. 1.5" x 1.5" each (e.g., installer-kit labels).

**Structure & Layers:**
The document is precisely structured with dedicated layers for production efficiency:
*   **`Artwork` Layer:** Contains all live vector data-merge fields and visual elements.
*   **`CutContour` Layer (Spot Color):** Dedicated, non-printing spot color layer (e.g., "CutContour" or "Thru-Cut") holding all vector die-cut paths for each decal.
*   **`Bleed` Layer:** Guides and visual markers indicating the required bleed allowance around each decal cell.
*   **`TrimMarks` Layer:** Standard printer's trim and registration marks, positioned for accurate sheet alignment and cutting.
*   **`Guides` Layer (Locked):** Informational guides for layout, spacing, and safe zones.

**Placeholders & Styling:**
*   **Vector Data-Merge Fields:** Each decal cell contains a unique, live vector data-merge placeholder, such as `<<large_mark>>`, `<<medium_mark>>`, and `<<small_mark>>`. These fields are configured to dynamically import and place vector assets (e.g., logos, QR codes, part numbers) from a data source directly into the corresponding decal cell.
*   **Bleed Allowance:** Each decal cell includes a standard 0.125" bleed beyond the cut line, ensuring no unprinted edges after trimming.
*   **Cut-Line Registration:** The vector cut-paths on the `CutContour` layer are precisely registered to their respective artwork and merge fields, guaranteeing accurate die-cutting of the merged artwork.
*   **Brand Styling:** Follows standard print production best practices, including clear layer naming, spot color definition for cut paths, and precise vector construction, ensuring compatibility with automated cutting equipment. The template is designed for direct integration with an external data source for mass customization.
