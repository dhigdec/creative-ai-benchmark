**Template Description: `open_house_finance_flyer.indd`**

This is a 2-page (front/back) InDesign template for an 'Open House Finance Flyer', formatted to US Letter dimensions (8.5" x 11").

**Front Page Structure:**
*   **Hero Image Frame:** A wide, top-aligned image placeholder (`HeroImage_Frame`).
*   **Design Element:** A decorative brass/gold horizontal rule below the hero image.
*   **Headline Area:** A prominent text frame for a main headline (`Headline_Text`).
*   **Branding:** A dedicated image frame for a company logo (`CompanyLogo_Frame`).
*   **Lender Profile:** An image frame for a lender's headshot (`LenderHeadshot_Frame`).

**Back Page Structure:**
*   **Payment Scenario Table:** A pre-formatted table designed to display financial scenarios, containing the following live InDesign data-merge fields:
    *   `<<PurchasePrice>>`
    *   `<<DownPaymentOptions>>`
    *   `<<MonthlyPayment>>`
    *   `<<AppreciationProjection>>`
*   **Property Information:** A text frame containing the data-merge field `<<PropertyAddress>>`.
*   **Contact Information:** A bottom strip featuring a QR code placeholder (`QRCode_Frame`) and contact details.

**Styling & Features:**
*   **Color Palette:** Incorporates brass/gold accents, suggesting a sophisticated and financial brand aesthetic.
*   **Placeholders:** All text and image frames are named for easy identification and content population.
*   **Data Merge Ready:** Utilizes genuine InDesign `<<field>>` placeholders for direct data binding via data merge.
