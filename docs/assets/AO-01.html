**Template Description: carton.indd**

**File Type:** Adobe InDesign document (.indd)

**Purpose:** Print-ready unit carton dieline template for product packaging.

**Layout:** Single-page flat dieline featuring 5 box panels (front, back, two sides, top).

**Dimensions & Print Specs:**
*   3mm bleed
*   CMYK color space
*   Includes registration and crop marks

**Content & Placeholders:**
*   **Front Panel:**
    *   Named picture frame: `HERO_PLATE` (pre-sized for finished white-background hero image).
    *   Placed vector logo: DERMAVERA.
    *   Live text frame(s): For SKU name.
*   **Back Panel:**
    *   Live text area: For INCI list and product claims.
    *   Reserved rectangular frame: For barcode placement.
*   **Other Panels:** Blank by default for structural integrity or unlisted content.

**Brand Styling:** Integrates the DERMAVERA brand logo.

**Usage:** Designed for direct export via `document_render_layout` to a print-ready PDF; does not support data merge.

**Template Description: label.ai**

This is a hand-authored Adobe Illustrator label template, 70x90mm with an additional 3mm bleed, set in CMYK color mode. The design features a static DERMAVERA vector logo positioned top-centre, a thin hairline rule, and a clean, clinical whitespace layout.

Dynamic content is driven by genuine Illustrator Variables (bound as Dynamic objects, not literal text), ensuring data merge compatibility. These variables include:
*   `{sku_name}` for the main display heading.
*   `{claim}` for a subhead.
*   `{volume}` placed in the lower-right corner.
*   `{inci_line}` as a small-print body block.
*   `{batch}` for footer micro-text.

A brand accent color is controlled by the `{accent_hex}` variable, which drives a specific swatch/text accent. All variable names (`sku_name`, `claim`, `volume`, `inci_line`, `batch`, `accent_hex`) directly correspond to expected CSV headers for `document_merge_data_vector` processing, enabling one label output per data row.
