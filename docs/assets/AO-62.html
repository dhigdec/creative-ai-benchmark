**Template Description: catalog_template.indd**

This is a client-supplied Adobe InDesign document (.indd/.idml) designed as a single A4 portrait (210x297mm) catalog card master, including a 3mm bleed.

**Structure & Layout:**
The template features a clean, single-page layout for individual product listings. Key elements are positioned to showcase product information and imagery.

**Placeholders:**
*   **Data Merge Text Fields:**
    *   `<<ProductName>>`
    *   `<<Blurb>>`
    *   `<<Price>>`
    *   `<<SKU>>`
    *   `<<EventName>>`
    *   *Critical Feature:* All variable text frames have "Auto-Size: Height Only / Shrink text to fit" enabled to prevent overset text with longer merged strings.
*   **Image Frames:**
    *   One dedicated frame for prepped product art.
    *   One dedicated frame for a background image.

**Brand Styling:**
The template provides a structured visual framework for catalog cards, leveraging InDesign's styling capabilities for consistent typography and layout. While specific fonts/colors are inherent to the .indd, the primary functional styling for data merge fields ensures content adaptability.

**Technical Notes:**
This is a genuine InDesign file with bindable data merge fields, essential for the `document_merge_data_layout` step. Field names are precisely matched to expected CSV column headers. The auto-fit text frames are a mandatory feature to manage dynamic content length.

**Template Description: `pricetag_template.ai`**

This is a client-supplied Adobe Illustrator (.ai) template designed for a 70x40mm shelf price tag.

**Structure & Layers:**
The template is a vector-based Illustrator document, implying distinct layers for background, text, and image elements. Its structure is optimized for dynamic content population via Illustrator Variables.

**Dimensions:**
70mm (width) x 40mm (height).

**Placeholders & Binding:**
The template features specific dynamic objects bound to Illustrator Variables, ensuring compatibility with data merge operations where variable names match CSV column headers:
*   `ProductName` (Text Placeholder): An area-type text object set with 'Auto Size' to automatically scale font size to fit longer product names within the designated area.
*   `Price` (Text Placeholder): A text object for displaying product pricing.
*   `SKU` (Text Placeholder): A text object for displaying the Stock Keeping Unit.
*   `Logo` (Image Placeholder): A linked-image variable placeholder for dynamically inserting a brand or product logo.

**Brand Styling:**
As an Illustrator document, the template inherently carries predefined brand styling including specific fonts, colors, and overall layout, designed to maintain consistent visual identity for the price tags.
