**Template Description: `verdafix_label_master.indd`**

This is a genuine Adobe InDesign master template for a Verdafix sealant-bottle wrap label, meticulously rebuilt to match the brand's existing legacy aesthetic and styling.

*   **Dimensions:** 90 x 130 mm (portrait orientation)
*   **Bleed:** 3 mm on all edges
*   **Color Intent:** CMYK
*   **Brand Styling:** The template faithfully replicates Verdafix's established visual identity, including existing typography, color palette, logo placement, and overall design layout, ensuring brand consistency with current product lines.

*   **Structure & Layers:** The document is structured with standard InDesign layers for clear content organization, typically including:
    *   `Background/Graphics`: For static design elements and brand patterns.
    *   `Images`: Dedicated frames for visual assets.
    *   `Text`: For all dynamic and static text elements.
    *   `Non-Printing Info`: For guides, crop marks, and internal notes.

*   **Data-Merge Placeholders:** The template features active InDesign data-merge fields for dynamic content population:
    *   `<<product_name>>`
    *   `<<variant>>`
    *   `<<net_content>>`
    *   `<<sku>>`
    *   `<<batch_prefix>>`
    *   `<<h_statements>>` (Hazard statements)
    *   `<<p_statements>>` (Precautionary statements)
    *   `<<usage_de>>` (German usage instructions)

*   **Image Placeholders:** Three dedicated image frames are pre-configured:
    *   Product photo
    *   GHS hazard pictogram
    *   EAN barcode
