**File:** `titanseal_carton_mechanical.ai`

**Description:**
Illustrator template for a corrugated shipping carton mechanical. Designed for single-color (K=100) black print on natural kraft corrugated stock, aligning with cost constraints.

**Structure & Variants:**
The template features two primary size variants – 20-cartridge and 24-tube – dynamically controlled via an Illustrator `CARTON_SIZE` visibility variable. This variable toggles between the respective die-line geometries and associated artwork layouts, ensuring accurate scaling and positioning for each carton type.

**Layers:**
Organized with a dedicated, non-printing, locked `_DIE_LINE` layer (utilizing a spot color, e.g., "Dieline" or "CutContour") and a primary `_ARTWORK` layer for all printable elements.

**Content & Brand Styling:**
Includes the TitanSeal mono mark logo, standardized "Fragile" and "This Way Up" pictograms, and brand-consistent typography for all text elements. All graphics are rendered in solid black.

**Illustrator Variables (Pre-defined):**
The template is authored with Illustrator Named Variables, enabling automated data merge.

*   **Text Variables:**
    *   `SKU_NAME` (e.g., "TitanSeal Silicone Sealant")
    *   `SKU_CODE` (e.g., "TS20CRTG-BLK")
    *   `PACK_COUNT` (e.g., "20", "24")
    *   `BATCH` (e.g., "20231015A")
*   **Visibility Variable:**
    *   `CARTON_SIZE` (Controls which carton variant's die-line and artwork group is active, e.g., "20-Cartridge" or "24-Tube").

**Functionality:**
Configured for direct data binding from a CSV (e.g., `document_merge_data_vector.csv`), allowing rapid generation of specific SKU carton artwork per row.

**Template File:** `titanseal_label_mechanical.indd`

**Type:** Adobe InDesign Data Merge Template

**Purpose:** Client-provided master template for generating TitanSeal brand wrap-around cartridge labels. Designed for automated population of product-specific data via InDesign's native Data Merge feature, linking directly to a CSV data source.

**Dimensions:**
*   **Primary Page Size:** 200mm (width) x 95mm (height)
*   **Bleed:** 3mm applied universally to all edges.
*   **Content Adaptability:** The layout is engineered to accommodate a 297mm tube length variant (maintaining the 95mm height), with flexible text frames and graphic zones designed to scale or re-flow appropriately without manual adjustment.

**Structure & Layers:**
The document utilizes a standard layered structure for print production:
1.  **Die-line/Cut:** Non-printing layer defining the precise label cut contour.
2.  **Base Graphics:** Static brand elements, background colors, and fixed design patterns.
3.  **Dynamic Content:** Contains all data-merged text frames and image placeholders.
4.  **Legal/Fine Print:** Fixed legal disclaimers, compliance text, and registration marks.

**Placeholders (Native InDesign Data Merge Fields):**
The template features the following *real* InDesign data-merge placeholder fields, designed to bind directly to corresponding CSV header columns:
*   `<<SKU_NAME>>`: For the full product name (e.g., "TitanSeal Pro Grout Sealant").
*   `<<SKU_CODE>>`: For the unique product SKU or internal code (e.g., "TSG-5000").
*   `<<VOLUME>>`: For the cartridge volume (e.g., "310ml", "10.5 fl oz").
*   `<<COLORWAY>>`: For the product's color description (e.g., "Clear", "Grey", "White").
*   `<<BARCODE>>`: For the barcode string (e.g., EAN-13, UPC). This field is typically styled with a barcode font or linked to an external barcode image generation process.
