**Template Description: `poster_template.indd`**

This is an Adobe InDesign (.indd) template for a single A2 portrait (420x594mm) fashion editorial poster, styled in a monochrome documentary/magazine aesthetic.

**Structure & Layout:**
The template features a dominant full-height image frame positioned for a 4:5 aspect ratio, designed to showcase a treated runway look. Below or adjacent to this, a dedicated headline block is provided for the editorial title. A separate look-credit block is included for detailed attribution. The layout incorporates subtle thin grid-line rules, contributing to the documentary feel, and a distinct yellow highlight block as part of the editorial system's branding. A specific placeholder area is designated for the vectorized brand wordmark.

**Layers & Placeholders:**
The document is authored with live InDesign data-merge fields to facilitate automated content population. These include:
*   `<<look_no>>`
*   `<<look_title>>`
*   `<<fabric_credit>>`
*   `<<stylist_credit>>`

**Brand Styling:**
The template adheres to a monochrome palette, accented by the editorial system's signature yellow highlight block. Thin grid-line rules reinforce the structured, magazine-like, and documentary aesthetic.
