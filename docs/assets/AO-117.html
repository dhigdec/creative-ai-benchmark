**Template File:** `marisol_hero_template.indd`

**Description:**
This is an Adobe InDesign document containing two artboards for the Marisol Cove homepage hero section:

1.  **Desktop Artboard:** 1920x720 px
2.  **Mobile Artboard:** 1080x1350 px

**Structure & Layers (per artboard):**
Each artboard features a layered composition designed for responsiveness and brand consistency:
*   **Background Image Frame:** A full-bleed image frame positioned behind all other elements, intended for a "graded coastal crop" visual.
*   **Logo Frame:** Anchored to the top-left corner of the artboard.
*   **Text Area:** Dynamically populated via real InDesign data-merge fields.

**Placeholders & Data-Merge Fields:**
*   `<<headline>>`: Placeholder for the primary hero text.
*   `<<tagline>>`: Placeholder for secondary hero text.
*   **Background Image:** Image frame for the coastal visual.
*   **Logo:** Image frame for the brand logo.

**Brand Styling & Layout:**
The design ensures a consistent brand presence and clear messaging:
*   The left third of both the Desktop and Mobile artboards is intentionally kept clear of the background image's focal subject.
*   This designated clear area is reserved for the `<<headline>>` and `<<tagline>>` text, along with the logo, forming a distinct "headline/logo lockup."
