Full client brief
The client, Shady, has a finished custom pendant — Sterling Silver 925, opening double doors on hidden hinges, a ruby locking element, Islamic geometric openwork, and Arabic calligraphy engravings — and wants to present and market it ("share similar work, estimated cost, and timeline"). He supplies three studio photographs of the actual piece (a front-on hero, a three-quarter open-doors view showing the ruby latch, and a macro of the engraved calligraphy) plus a jeweler-authored InDesign spec/lookbook sheet (.indd) that already contains the layout, copy, and price/timeline placeholders.

Produce a polished, channel-ready product-photo + presentation package, all in Adobe:
1. Ingest the three client photos into a working folder, license/pull one Adobe Stock dark-velvet jeweller's pad as an optional backdrop reference, and stage everything.
2. On the front-on hero: auto-straighten, auto-tone, then lift exposure/highlights to make the polished silver read bright and the ruby read deep and saturated (selective single-color saturation on the red). Warm the white balance slightly for a luxury feel.
3. Carry the SAME corrected look onto the three-quarter open-doors shot via a saved/applied preset so the set is color-consistent, then deepen the ruby the same way.
4. Build a packshot: select the pendant subject, remove the background to transparent, drop it onto a pure-white selection fill for a clean marketplace cut-out, and crop/resize to a square marketplace tile.
5. Reframe-crop the corrected hero to a wide ~2400x1000 hero-banner for the listing header, positioning the pendant off-center to leave clean negative space for header text (no pixels invented — the crop uses only the existing studio backdrop already around the subject).
6. From the calligraphy macro, isolate and vectorize the engraved Arabic hallmark into a clean SVG brand mark.
7. Assemble the finished shots into a Firefly Boards moodboard the client can review.
8. Export the jeweler-authored InDesign spec/lookbook .indd to a print-ready PDF and a web JPEG.
Deliverables: corrected hero, color-matched open-doors shot, white-bg square packshot, wide hero-banner, ruby-deep variant, calligraphy SVG mark, Firefly board deep-link, and the exported spec-sheet PDF + JPEG.

It exercises a deep linear grade chain (auto-straighten → auto-tone → exposure → highlights → temperature → single-color saturation), a non-trivial color-MATCH requirement across two differently-cast shots via a saved preset, a multi-step masking branch (select_subject → remove_background → invert → solid fill → crop) where the agent must order the invert/fill correctly, the single legal generative step (reframe-crop), a raster→vector conversion, a moodboard assembly that reuses four derived outputs, and an export-only document step on a genuinely-authored input.
