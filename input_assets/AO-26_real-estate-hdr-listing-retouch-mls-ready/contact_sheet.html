Full client brief
You're the agency's go-to editor. We shot a listing and are sending you the frames; you handle the editing and return MLS-ready images with one consistent style across the whole property. For this batch, work the three representative frames we've attached (exterior hero + two interiors) end-to-end, then hand back a uniform set.

EXTERIOR HERO (wide dynamic range, blown-out windows and bright sky over a shaded facade and front lawn):
1. Auto-straighten the frame — the verticals are leaning and the horizon is off.
2. Crop to clean bounds after straightening to remove the skewed edges (this is our lens-distortion/alignment cleanup).
3. Recover the overall tone so the shaded facade reads properly — apply auto-tone as the base.
4. WINDOW PULL: select the bright windows and pull their highlights down so the glass shows interior/structure instead of blowing out; also tame the overall highlights.
5. SKY: select the sky and enhance it — recover the blown highlights and deepen the blue (we are NOT swapping the sky, just making the existing one read well and MLS-safe).
6. LAWN ENHANCEMENT: select the front lawn/grass and lift its green saturation and luminance so it looks healthy and inviting.
7. Warm the white balance very slightly and add a touch of vibrance so the whole exterior feels bright and welcoming.

STYLE LOCK + CONSISTENCY:
8. Pull the current Lightroom preset library so we can pick the house style.
9. Apply our chosen consistent "bright & airy real-estate" preset to the finished exterior — this is the look every frame in the property must match.
10-11. Apply that SAME preset to interior frame 1 and interior frame 2, then bright-window pull / highlight-tame each interior so the windows aren't blown and the rooms feel open — giving us one uniform style across the property.

DELIVERY: return each finished frame as a high-quality PNG, style-matched so the three read as one consistent property look, before you do the rest of the 20-40 on this listing. Turnaround 24-48h, bulk per-listing.

Workflow — 20 tool steps
  1. asset_add_file — Client supplies RAW/bracket-source frames from their local machine; open the Adobe file picker to bring the wide-DR exterior hero in and get its presignedAssetU
  2. image_auto_straighten — Level the leaning verticals/tilted horizon — the 'vertical/horizontal alignment' ask.
  3. image_crop_to_bounds — Crop away the skewed edges left by straightening — the alignment/lens-cleanup deliverable. Geometric crop only.
  4. image_apply_auto_tone — Base tonal recovery so the shaded facade reads — the single-file stand-in for HDR bracket blending.
  5. image_select_by_prompt — Mask the bright/blown windows for the window pull.
  6. image_adjust_highlights — WINDOW PULL: pull highlights down inside the window mask so glass shows structure instead of blowing out; same tool also tames overall highlights.
  7. image_select_by_prompt — Mask the sky for the MLS-safe sky ENHANCEMENT (not a replacement — replacement is not available).
  8. image_adjust_single_color_saturation — Deepen the blue and recover the sky within its mask so the existing sky reads cleanly.
  9. image_select_by_prompt — Mask the front lawn/grass for the lawn-enhancement ask.
  10. image_adjust_hsl — LAWN ENHANCEMENT: lift green saturation + luminance within the lawn mask so it looks healthy.
  11. image_adjust_color_temperature — Slight global white-balance warm-up for a bright, welcoming exterior. (Vibrance handled in next adjust call / preset.)
  12. image_adjust_vibrance_and_saturation — Touch of global vibrance to finish the welcoming exterior tone before the style lock.
  13. image_list_presets — Pull the Lightroom preset library to pick the locked house style (bright & airy). Lookup step — consumed by the apply calls below.
  14. image_apply_preset — Apply the chosen consistent bright-&-airy preset to finish the exterior hero — this output is the canonical look every other frame must match.
  15. asset_add_file — Bring in interior frame 1 (living room) from the picker. Secondary asset load for the consistency pass.
  16. image_apply_preset — Apply the SAME locked preset to the living room for one consistent property style.
  17. image_adjust_highlights — Tame the bright/blown living-room windows (interior window pull) to finish frame 1.
  18. asset_add_file — Bring in interior frame 2 (kitchen) from the picker. Secondary asset load.
  19. image_apply_preset — Apply the SAME locked preset to the kitchen — uniform style across all three frames.
  20. image_adjust_highlights — Tame the bright kitchen window to finish frame 2.
