Full client brief
Northgate Dental Students' Congress 2027 is a two-day event bringing roughly 1,000 delegates from dental schools across the country together for speakers, workshops, a gala dinner and socials. Our brand identity is already signed off, and we now need the physical delegate name badges produced from our attendee list — one personalised badge per delegate. We have handed off everything the job needs: our approved event wordmark as a flat high-contrast PNG (northgate_wordmark.png); an authored InDesign badge template (badge_template.indd) laid out at 90x130mm portrait with 3mm bleed, containing real InDesign Data Merge fields (<<FullName>>, <<School>>, <<Role>>, <<QRPath>>), a placeholder frame for the vectorized wordmark, a full-width role colour-band frame, and a QR image frame; our delegate roster export (delegate_roster.csv) with one row per delegate holding full name, school, role (Delegate / Speaker / Committee / Press) and the matching role-QR filename; and four flat role-band QR marks as PNGs (one per role, each a solid-colour block with the check-in QR). The wordmark PNG is slightly oversized and off-centre for the badge frame and needs cleaning and vectorizing so it stays razor-sharp at print size; the four QR PNGs are inconsistent sizes and must be squared and cropped to the badge's QR frame. The job: (1) crop/clean the wordmark and square-crop the four role QRs to the exact frame ratios the template expects; (2) vectorize the flat wordmark into a clean SVG so it prints crisp on ~1,000 badges; (3) upload all prepped assets to our workspace; (4) run the InDesign data-merge so every roster row binds its name, school, role band and role QR into the authored template — one badge record per delegate; (5) export the merged document to a print-ready imposed PDF (90x130mm + bleed, ~300 DPI, crop marks) for the lanyard printer. Roles, names and schools all come from the roster as-is — nothing is invented; long names must bind into the template's auto-fit name frame without clipping.

  • northgate_wordmark.svg — Clean vector SVG of the approved event wordmark produced by image_vectorize from the cropped wordmark PNG, placed in the badge template so the mark stays razor-sharp across all ~1,000 printed badges.

**File:** `badge_template.indd`

**Purpose:** Adobe InDesign master template for a single 90x130mm portrait conference delegate name badge, ready for data merge.

**Dimensions:** 90mm (W) x 130mm (H) portrait, with 3mm bleed on all sides.

**Structure & Layers:**
1.  **Top:** Vectorized event wordmark image frame.
2.  **Mid-Upper:** `<<FullName>>` text frame.
3.  **Mid-Lower:** `<<School>>` text frame.
4.  **Lower:** `<<Role>>` text frame.
5.  **Behind <<Role>>:** Full-width rectangular frame for a role-specific color band.
6.  **Lower Right:** Square image frame for QR code.

**Placeholders (Data Merge Fields):**
*   **Text Fields:**
    *   `<<FullName>>`: Configured with Auto-Size 'shrink text to fit' to prevent overset text for long names.
    *   `<<School>>`
    *   `<<Role>>`
*   **Image Field:**
    *   `<<QRPath>>`: Populates the lower-right QR code image frame, wired to a data source column containing image file paths (e.g., `qr_john_doe.png`).

**Brand Styling & Functionality:**
*   Distinct visual separation for delegate role via a full-width color band.
*   Professional event branding via a dedicated wordmark placement.
*   Dynamic, personalized QR codes for each delegate.
*   Robust text handling for `<<FullName>>` ensures legibility regardless of name length.
*   Authored as a genuine InDesign document with live data merge functionality, ensuring proper binding of text and image fields from a CSV data source.
