Full client brief
Lumen Quarterly is a fictional independent arts-and-culture magazine published by Lumen House Press. Our in-house junior designer has finished laying out the Autumn 2026 issue in InDesign and exported it as a print proof PDF, but before it goes to the printer we need a fresh set of eyes to check that the whole issue actually stays inside our brand framework — our board is strict about consistency and the last issue went out with an off-palette pull-quote and a stretched masthead lock-up that we only caught in print. You will receive: (1) lumen_autumn2026_proof.pdf, the full 16-page laid-out issue (cover, masthead, contents, editor's letter, three feature spreads, department pages, back-cover ad); (2) lumen_brand_styleguide.pdf, our complete brand style guide covering the approved colour palette (Lumen Ink #1B2A4A, Signal Coral #FF5A47, Paper #F4F1EA — with the exact hex values and the rule that body text is Lumen Ink only), the masthead / logo lock-up rules (clear-space, minimum size, never stretched or recoloured), the typography system (Freight Display for headlines, Untitled Sans for body, specified sizes and tracking), and the imagery tone rules (warm, editorial, consistent grade; no cold or oversaturated photos); and (3) two flat reference images — the approved palette swatch strip and the correct masthead lock-up — to compare on-page elements against. The job is an audit, not a redesign — do NOT re-lay-out, restyle, recolour or edit anything. Go through every page, compare what is on the page against the guideline, and hand back a brand-consistency audit report that, for each issue found, gives the page number, the on-page region, a plain-language description of the drift, the specific guideline clause it violates, the approved value versus the observed value, and a severity (blocker / warning / nit). We expect you to catch things like: a headline set in the wrong weight or the wrong face, a coral used at the wrong hex, the masthead placed with too little clear-space or subtly stretched, a feature photo that reads cold/blue against the warm-editorial imagery rule, and body copy set in a non-approved colour. Deliver the findings as a clean marked-up report file plus the raw findings as a structured data file, and open with a one-page cover summary giving the total count by severity. Page size is A4 portrait; the report is for the editor and the board, not the printer.
