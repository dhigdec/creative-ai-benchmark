Full client brief
LUMIÈRE MÉNAGE is a fictional modern clean-beauty and skincare label preparing to launch across social, e-commerce, unit cartons, and large in-store signage. After an internal round we have approved ONE monochrome logo concept and now need it PRODUCTIONIZED into a fully editable, reversible brand-mark system — this is a finishing/assembly job, not a from-scratch design round.

We supply a single flat, high-contrast raster export of the approved concept: a crisp geometric monogram ("LM" resolved into an abstract, symmetric negative-space shape) sitting above the wordmark "LUMIÈRE MÉNAGE" set in a clean modern condensed sans, rendered pure black on a near-white background, no gradients, no illustration, no flourishes. The concept scan is slightly skewed and the background is an off-white (#F5F5F3), not true white.

Deliverables, all chained in the Adobe connector:
1) Ingest our supplied concept PNG and get a clean working copy — straighten the slight skew and auto-tone so the black is dense and the edges are true.
2) Knock the off-white background out to true transparency so the trace has no halo/fringe.
3) Vectorize the cleaned lockup into a clean, fully editable SVG — this is the MASTER vector (basis for the requested AI/SVG editable file + high-res PNG exports).
4) Produce the REVERSED knockout: a white-on-dark version of the mark that reads perfectly when reversed out on dark packaging and signage (the brief's key requirement).
5) Derive a tight AVATAR / social-PFP version from the monogram alone (square, centered) that stays recognizable at avatar size.
6) Build a STACKED lockup crop for narrow / vertical placements.
7) Produce a ONE-COLOR pure-black print PNG (for single-color foil/deboss and small-format print).
8) Produce a single MONOCHROMATIC brand-tint alt (one hue only — no multicolor, honoring "clean monochrome palette") for tonal applications.
9) Recommend a brand font family + heading/supporting pairing that matches the modern condensed wordmark tone.
10) Emit a MONOCHROME palette recipe (black / dark grey / off-white / paper hex callouts) as the color-palette deliverable.
11) Assemble every produced variation into a usage board deep-link the founder can review and sign off in one place.

Hard constraints (honor the brief AND connector ground truth): stay strictly monochrome — no gradients, no multicolor, no vintage/distress treatment. Everything must reverse cleanly, scale from a social-media avatar up to large signage, and arrive as an editable vector plus high-res PNGs. No from-scratch logo drawing, no generative fill, no AI object removal, no compositing — the supplied concept is the single source of truth and every deliverable is derived from it by chaining. The requested AI/EPS export and the printed spec sheet are human/format-exp
