Full client brief
HaulAway (junk removal & cleanout, Maitland FL / North Orlando) supplies ONE existing logo they love but only as a flat, low-resolution raster PNG (a bold "HA" monogram lockup with the company wordmark beneath, in their black / lime-green / gray palette). They need it turned into a professional, print-ready brand identity kit usable on t-shirts, vehicle & trailer wraps, car magnets, and print.

The freelancer must drive the Adobe Creative Cloud connector end-to-end:
1) Ingest the client's raster logo and get a clean working copy (auto-tone + straighten the slightly skewed scan so edges are true).
2) Clean the color: neutralize any off-cast and crisp up the lime green and the blacks so the trace is sharp.
3) Clean-knock the background to transparent so the trace has no halo.
4) Vectorize the cleaned full lockup into a clean SVG = the MASTER vector (basis for AI/EPS/SVG/PNG exports).
5) Isolate the "HA" monogram for the ICON-ONLY mark: select the wordmark text by prompt, invert to keep just the monogram, solid-fill the wordmark region away with a solid white fill, then vectorize the isolated mark.
6) Produce the ONE-COLOR versions: a black-on-light knockout (solid black overlay) and a white-on-dark knockout (white fill) of the mark.
7) Build the layout LOCKUPS by cropping the cleaned master to its true bounds for the horizontal version and a tighter square for the stacked/vertical and the magnet.
8) Create the creative bonus "trailer-wrap hero": recolor the mark to a high-impact lime treatment, then set the surrounding canvas to solid black so the mark reads lime-on-black across a trailer side.
9) Identify/recommend the brand font family + a usage pairing (font_recommend) as the typography guide, and deliver the black/lime/gray hex/CMYK/Pantone callouts as the accompanying brand color reference.

Hard rules honored: ABSOLUTELY no generative fill, no AI object removal, no text-to-image, no compositing, no layout composition; image_fill_area is solid-color only; the master export step only EXPORTS genuine vector — the deliverable vector emerges from image_vectorize, not from an authored .ai. Every step after ingest consumes a prior step's output.

Workflow — 18 tool steps
  1. asset_initialize_file_upload — Ingest the client's supplied raster logo into CC storage and get a working asset reference/URI.
  2. asset_finalize_file_upload — Finalize the upload to a stable presigned asset URI that the image_* tools can consume.
  3. image_auto_straighten — Correct the ~2.5° scan skew so the logo edges are true before tracing — straight paths trace cleaner.
  4. image_apply_auto_tone — Auto-tone the straightened logo to normalize exposure/contrast and recover clean black/white separation.
  5. image_adjust_single_color_saturation — Boost only the lime-green channel so the brand green is punchy and clean-edged for the trace (other colors untouched).
  6. image_remove_background — Knock the near-white #F4F4F2 background to true transparent so the vectorize has no halo/fringe.
  7. image_vectorize — Vectorize the cleaned full lockup → MASTER print-ready SVG (deliverable; basis for AI/EPS/PNG).
  8. image_select_by_prompt — Select the 'HaulAway' wordmark + tagline text by prompt so it can be removed to isolate the monogram.
  9. image_invert_selection — Invert the wordmark selection so the kept region is just the 'HA' monogram mark.
  10. image_fill_area — Solid-fill the wordmark area with solid white (solid color only) leaving the isolated HA monogram.

  15. image_crop_and_resize — Re-crop/resize to a square-ish frame → stacked/vertical layout version for magnet + left-chest (deliverable).

Why it's non-trivial
