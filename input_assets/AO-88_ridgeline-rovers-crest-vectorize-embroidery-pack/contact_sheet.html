Full client brief
Ridgeline Rovers FC, an amateur mountain-town football (soccer) club, is ordering embroidered crest patches and sleeve badges from a factory and needs its emblem turned into proper production-ready vector artwork first. The only artwork the club has is a phone photo of a sample patch that a previous designer stitched off an early draft (crest_sample_patch.jpg) — a shield-shaped crest with a stylized mountain-ridge silhouette, a football, the letters "RRFC", and the founding year "EST. 2014" on a banner ribbon, stitched in four thread colors (forest green, cream, slate grey, black outline) on a heathered fabric backing. The photo is a real hand-held shot: tilted about 3 degrees, dull and slightly underexposed, a cool/green color cast from the fabric, and the twill weave texture shows across every color so the edges look fuzzy. That texture and cast make it useless for a factory as-is.

The job: (1) clean the reference so it can be traced — deskew it level, auto-tone to fix the flat exposure, neutralize the green fabric cast so the four thread colors read true, then push contrast hard to crush the woven-fabric texture and drive every element toward a clean, flat, high-contrast silhouette; knock the heathered backing to transparent so the trace has no halo. (2) VECTORIZE the cleaned crest into a clean, scalable SVG — this is the master emblem, and it must stay razor-sharp when the factory scales it to the requested medium 3-6 inch patch size. (3) Because embroidery and woven patches need each color as one flat solid thread, reduce the emblem to FLAT SPOT SEPARATIONS: go element by element, select each color region, and lock it to a single solid ink from the club's 4-color palette (no gradients, no anti-aliased blends), then pull a single-color BLACK OUTLINE plate the factory uses for the merrowed-border stitch path, and re-vectorize the flattened art. (4) Deliver a one-color KNOCKOUT (solid emblem silhouette) the factory can use to cut the twill/felt backing. (5) Recommend a clean condensed athletic font for the "RRFC" / "EST. 2014" lettering so the club can extend the wordmark later, and write it up alongside the exact thread colors as a reference sheet. (6) Export the artwork to vector masters (PDF/PNG/SVG) from a genuine authored .ai and gather everything into a shared review folder so the club can approve the crest and the exact thread colors before the factory runs it.

Palette: forest green #2E5A3A, cream #F2E9D0, slate grey #6B7078, black #111111 outline. Deliverables: master crest SVG, flat spot-separated SVG, black-outline plate, one-color knockout, exported vector masters, a stitch-color / thread reference sheet, and a shared review-folder link. This Adobe Illustrator (.ai) document serves as the production-ready vector master for the Ridgeline Crest emblem, intended primarily for patch manufacturing. It features vector geometry hand-assembled from the traced `ridgeline_crest_master.svg` and its 4-flat-ink spot-separated counterpart, `ridgeline_crest_spot_separated.svg`.

The file contains multiple artboards, each precisely sized for typical 3-6 inch patch dimensions (e.g., 4x4 inches). No new artwork is drawn within this file; its purpose is to consolidate and arrange the traced paths for final export.

**Layers & Placeholders:**
The artwork is structured with vector paths organized for clean output and color separation. The emblem design is complete, and the file contains no empty placeholders or editable text fields for new content.

**Dimensions:**
Artboards are configured for physical output dimensions ranging from 3 to 6 inches, suitable for various patch sizes.
